## 🚀 Overview

**CodePulse** is a feature-rich, high-performance web application designed to help first-year students and beginner developers build a consistent coding habit. It gamifies the practice experience with streaks, XP, levels, and a global leaderboard, making the journey from "Hello World" to mastery engaging and transparent.

> "The only way to learn a new programming language is by writing programs in it." — *Dennis Ritchie*

---

## ✨ Key Features

### 📊 Comprehensive Dashboard
- **Streak Tracking**: Stay motivated with daily streak counters and "Miss Me?" reminders.
- **Dynamic Stats**: Real-time tracking of problems solved, total hours, and consistency scores.
- **XP & Levels**: Earn XP for every session and level up from a "Beginner" to a "Legend".
- **Daily Challenges**: Personalized tasks to keep your skills sharp.

### 📝 Smart Logging
- **Session Tracking**: Log topic, difficulty, time spent, and mood.
- **Reflection Notes**: Add notes to your sessions to reinforce learning.
- **XP Previews**: See exactly how much XP you'll earn based on session difficulty and timing.

### 📈 Deep Analytics & Progress
- **Academic Journey**: Visualizes your progress over time with interactive charts (powered by Chart.js).
- **Topic Breakdown**: Donut charts showing your skill distribution across categories.
- **Consistency Heatmap**: A GitHub-style contribution grid to see your long-term activity.
- **Learning Path**: A guided roadmap from Arrays to Dynamic Programming.

### 🏆 Gamification & Community
- **Global Leaderboard**: See where you stand among your peers.
- **Achievement Badges**: 15+ unique badges to unlock as you hit milestones.
- **Study Buddy**: Get matched with other learners for accountability.
- **Shoutout Wall**: celebrate small wins with the community.

### 🌓 Premium UI/UX
- **Dark & Light Mode**: Seamless theme switching for comfortable coding.
- **Responsive Design**: Works perfectly on both desktop and mobile devices.
- **Micro-interactions**: Smooth transitions and elegant animations using CSS and JS.

## 🛠️ Tech Stack

- **Frontend**: 
  - Semantic HTML5 & Modern Vanilla CSS
  - Vanilla JavaScript (ES6+)
  - [Chart.js](https://www.chartjs.org/) for animations and data visualization.
- **Backend**:
  - [Node.js](https://nodejs.org/) & [Express.js](https://expressjs.com/)
  - RESTful API architecture
- **Database**:
  - `db.json` local storage (file-based persistence)
  - Seamless LocalStorage sync for offline/online experience.

## ⚙️ Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your system.

### Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/Beginner-Coding-Practice-Tracker.git
   cd Beginner-Coding-Practice-Tracker
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the server**:
   ```bash
   npm start
   ```

4. **Access the application**:
   Open your browser and navigate to `http://localhost:3001`.
